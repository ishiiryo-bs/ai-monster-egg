# AI Monster 卵診断

あなたのAI活用タイプから、世界に一つの卵が生まれるWeb診断。

👉 **診断ページ: https://icryodev.github.io/ai-monster-egg/**

## 診断から来た方へ

あなたの卵は「引きつぎコード」で待っています。

- macOS版アプリ(AIを使うほど育つデスクトップの相棒)が公開されたら、コードを入力するとその卵をそのまま育てられます
- コードを控えておけば、卵が消えることはありません
- 公開の知らせは X: [@icryodev](https://x.com/icryodev) からお届けします

## 開発メモ

React + TypeScript + Vite。GitHub Pagesへ自動デプロイ。

```bash
npm install
npm run dev     # 開発サーバ
npm run build   # ビルド
```
